# Catmash

Available on : <https://maxrob-catmash.herokuapp.com/catmash>
Catmash api on Github : <https://github.com/maxrob/catmash-api>

### Install Project:

```Bash
npm install
```

## Client :

### Run app in the development mode

```Bash
npm start
```

### Launch tests

```Bash
npm test
```

### Builds app for production

```Bash
npm run-script build
```
